# ФинГоризонт - SaaS сервис финансового прогнозирования

**Слоган:** «Планируйте уверенно»

## Описание проекта

ФинГоризонт — это веб-сервис для финансового планирования и прогнозирования на основе исторических данных. Система позволяет создавать сценарии бюджета, вносить фактические данные и получать прогнозы на будущие периоды.

## Технический стек

- **Frontend:** HTML5, CSS3, Vanilla JavaScript
- **Backend:** PHP 8.x
- **База данных:** MySQL
- **Сервер:** Apache
- **Графики:** Chart.js

## Установка

### 1. Требования

- PHP 8.0 или выше
- MySQL 5.7 или выше
- Apache с mod_rewrite

### 2. Настройка базы данных

```bash
# Импортируйте схему БД
mysql -u root -p < sql/schema.sql
```

### 3. Конфигурация

Отредактируйте файл `config/database.php`:

```php
define('DB_HOST', 'localhost');
define('DB_NAME', 'finhorizon');
define('DB_USER', 'ваш_пользователь');
define('DB_PASS', 'ваш_пароль');
```

### 4. Настройка Apache

Создайте виртуальный хост:

```apache
<VirtualHost *:80>
    ServerName finhorizon.local
    DocumentRoot /path/to/finhorizon/public
    
    <Directory /path/to/finhorizon/public>
        AllowOverride All
        Require all granted
    </Directory>
</VirtualHost>
```

## API Endpoints

### Сценарии (`/api/scenarios.php`)

| Метод | Action | Описание |
|-------|--------|----------|
| GET | list | Список сценариев пользователя |
| GET | get?id=X | Получение сценария по ID |
| POST | create | Создание нового сценария |
| POST | update?id=X | Обновление сценария |

|-------|--------|----------|

| GET | items?scenario_id=X | Статьи бюджета |
| GET | summary?scenario_id=X | Сводные данные |
| GET | forecasts?scenario_id=X | Прогнозы |
| POST | category | Создание категории |
| POST | item | Добавление статьи |
| POST | generate_forecast?scenario_id=X | Генерация прогноза |

## Алгоритм прогнозирования

Система использует метод **скользящего среднего**:

1. Берутся последние 3 периода с данными
2. Вычисляется среднее арифметическое
3. Рассчитывается волатильность для определения уровня доверия
4. Прогноз генерируется на N периодов вперед

Формула:
```
Прогноз = (Значение[n-1] + Значение[n-2] + Значение[n-3]) / 3
Уровень доверия = 100 - (Стандартное отклонение / Среднее * 100)
```
